## :computer:How to use

### Online use
[https://logo-maker.djfos.fun](https://logo-maker.djfos.fun)

### Locally

1. Clone this repository

```bash
git clone https://github.com/reamd/logo-maker.git
cd logo-maker
```

2. Install dependencies

```bash
npm install
```

3. Start web-application

```bash
npm run dev

http://localhost:5173/
```
